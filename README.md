# 🌿 Wildkeep — A Field Guide to the Animal Kingdom

Wildkeep is a browsable field guide concept for the animal kingdom, catalogued the way a naturalist would keep their notes — classification, habitat, diet, and conservation status, with one memorable fact per entry.

**🔗 Live demo:** https://nibras-02.github.io/wildkeep-field-guide/

---

## ✨ Features

- **Browse by class** — filter specimens by Mammalia, Reptilia, Aves, Amphibia, Insecta, and Actinopterygii
- **Search** — matches species name, scientific name, habitat, or diet
- **Sort** — by name (A–Z), conservation status, or class
- **Conservation status filter** — narrow the archive to Endangered, Vulnerable, Near Threatened, or Least Concern
- **Favourites** — star any specimen to save it to your session favourites, with a live count
- **Surprise Me** — pulls a random specimen from the current filtered results and opens its detail view
- **Dark / Light mode** — full theme toggle, not just a colour swap
- **Detail view** — click any card for an expanded profile with habitat, diet, status, and a notable fact

## 🛠️ Built With

- HTML5, CSS3 (custom properties for theming), vanilla JavaScript — no frameworks, no dependencies
- Google Fonts: Fraunces, Inter, IBM Plex Mono

## 🚀 Running Locally

No build step required — it's a single static file.

```bash
git clone https://github.com/Nibras-02/wildkeep-field-guide.git
cd wildkeep-field-guide
```

Then just open `index.html` in your browser.
